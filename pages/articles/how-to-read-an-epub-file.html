How to Read an ePub File

The ePub format is a widely used format for digital books. I have an ePub version of the RCV on the Download page. Below, I will list a few of my recommended ePub readers for reading ePub books on certain devices. This post is quite limited since it would be near impossible for me to cover every application and device that is ePub compatible. This list will consist of applications I have used or still use on devices I own. A much larger list of compatible applications and devices can be found on the ePub article at Wikipedia.

ePub Reader Applications

  • For Mac OS X and Windows computers: Adobe Digital Editions, Calibre, FBReader
  • For Linux systems: Calibre, FBReader
  • For most desktop browsers (Firefox, Chrome, Vivaldi, Opera, Edge): EPUBReader (extension)
  • For the iPad, iPhone, and iPod Touch: iBooks
  • For Android: FBReader, Librera
